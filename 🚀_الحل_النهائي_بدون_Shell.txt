الطريقة 2️⃣: إعادة إنشاء الخدمات (إذا لم تنجح الطريقة الأولى)
═══════════════════════════════════════════════════════════════

إذا استمرت المشكلة، احذف الخدمات القديمة وأنشئ جديدة:

Backend:
────────
1. اذهب إلى Render Dashboard
2. احذف الـ Backend Service القديم
3. اضغط "New +" → "Web Service"
4. اختر الـ Repository: -ecom-parent-project
5. املأ البيانات:
   - Name: ecom-backend
   - Root Directory: backend
   - Build Command: chmod +x build.sh && ./build.sh
   - Start Command: gunicorn ecom_project.wsgi:application
   - Environment: Python 3
6. أضف Environment Variables من ملف .env
7. اضغط "Create Web Service"

Frontend:
─────────
1. اذهب إلى Render Dashboard
2. احذف الـ Frontend Service القديم
3. اضغط "New +" → "Static Site"
4. اختر الـ Repository: -ecom-parent-project
5. املأ البيانات:
   - Name: ecom-frontend
   - Root Directory: frontend
   - Build Command: npm install && npm run build:render
   - Publish Directory: build
6. اضغط "Create Static Site"
